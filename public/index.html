Routes for this API:


/login : page to log in to account, GET-method
/login/signup : page to create new account
/login/login : endpoint for login requests, POST-method
/login/upload : page to upload files
/login/logout : post endpoint to logout, POST-method
/resources : returns a list of files that are saved on this webserver and their respective owner
/resources/id : download a file with the given id (must be logged in), GET-method
/create : add a file onto the server, POST-method
/update : update a file using the file id (found using /resources), POST-method
